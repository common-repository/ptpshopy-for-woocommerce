== Description ==
[PTPShopy](https://ptpshopy.com/) is cryptocurrency payment gateway, which allows users to checkout with popular cryptocurrencies.

How to setup PTPShopy on the WooCommerce webstore:
* Download and Activate PTPShopy For WooCommerce.
* Enable from WooCommerce > Settings > Payments > PTPShopy. 
* Enter API Key from Manage.
* It will automatically appear to the customer on checkout.

After the installation you will be able to use pay with ptpshopy.com automatically.
Visit our [Integration Guide](https://ptpshopy.com/knowledge-integration-guide/) or [FAQ Page](https://ptpshopy.com/faq/) for other details.
